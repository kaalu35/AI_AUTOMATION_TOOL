The Demo Web Shop application (https://demowebshop.tricentis.com/) is a web-based e-commerce platform that supports both guest and registered users, enabling them to browse product categories such as Books, Computers, Electronics, Apparel & Shoes, Digital Downloads, Jewelry, and Gift Cards, search for products using a keyword-based search bar, and view product details including name, description, price, and quantity selection. Users can register by providing mandatory details such as first name, last name, valid email, and password with confirmation, and log in using valid credentials (e.g., [testuser1@test.com](mailto:testuser1@test.com) / Test@123), with proper validation and error handling for invalid inputs. Once authenticated, users can add products to the shopping cart, update quantities, remove items, and view subtotal calculations. The application supports a multi-step checkout process including billing address, shipping address, shipping method, payment method, and order confirmation, where all mandatory fields must be validated before order placement. Users can also view their order history and details under the orders section. The system ensures proper session management (login/logout), displays meaningful error messages for invalid operations (such as login failure or empty cart checkout), and meets non-functional requirements including secure HTTPS communication, responsive UI, fast page load performance (less than 3 seconds), and usability through intuitive navigation and clearly identifiable UI elements
